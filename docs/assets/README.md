# Assets Directory

This directory contains template assets and placeholder files.

## Structure

- **professional/**: Your professional photos, videos, and work-related images
  - Add your profile image/video here
  - Add your logo or brand images
  - Add screenshots of your work

- **favicon/**: Favicon files for your site
  - favicon.ico
  - site.webmanifest
  - Various icon sizes for different devices

## Template Instructions

1. Replace the placeholder files with your actual assets
2. Update file paths in mkdocs.yml to match your asset filenames
3. Optimize images for web (compress, resize appropriately)
4. Ensure favicon files are properly formatted

## Recommended Image Sizes

- **Profile Image**: 400x400px (square, for consistent display)
- **Logo**: 200x200px (transparent background preferred)
- **Favicon**: 32x32px (ICO format)
- **Blog Images**: 1200x630px (for social media sharing)

## File Formats

- **Images**: JPG (photos), PNG (logos/graphics with transparency), WebP (modern browsers)
- **Videos**: MP4 (H.264 codec for best compatibility)
- **Icons**: SVG (scalable), PNG (fallback)
