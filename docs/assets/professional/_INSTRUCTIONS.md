# Professional Assets Directory

This directory contains placeholder assets for the template. Replace these files with your actual professional assets:

## Files Created:

### Favicon Assets (`favicon/`)
- `favicon.ico` - Website icon for browser tabs and bookmarks
- `site.webmanifest` - Web app manifest for PWA functionality

### Professional Assets (`professional/`)
- `logo.jpg` - Your personal or company logo
- `profile.jpg` - Profile photo for about page and general use  
- `avatar.jpg` - Avatar image for blog author profile
- `profile_video.mp4` - Optional profile video for homepage

## Instructions:

1. **Replace placeholder files** with your actual assets
2. **Keep the same filenames** or update the references in your content files
3. **Optimize for web**: Compress images and videos for faster loading
4. **Recommended sizes**:
   - Favicon: 32x32px (ICO format)
   - Logo: 200x200px (PNG with transparency preferred)
   - Profile/Avatar: 400x400px (square aspect ratio)
   - Video: 720p or lower for web optimization

## File Formats:
- **Images**: JPG (photos), PNG (logos/graphics with transparency)
- **Video**: MP4 (H.264 codec for best compatibility)
- **Favicon**: ICO format for maximum browser support
